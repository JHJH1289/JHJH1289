# 안녕하세요, 개발자 정재훈입니다.

React를 중심으로 웹 프론트엔드를 학습하고 프로젝트로 구현하고 있습니다.  
사용자가 자연스럽게 이해할 수 있는 화면, 읽기 쉬운 코드, 꾸준한 기록을 중요하게 생각합니다.

## Tech Stack

**Frontend**  
React, JavaScript, HTML5, CSS3

**Backend**  
Java, Spring, Node.js, REST API

**Database**  
Oracle, PostgreSQL, JPA
